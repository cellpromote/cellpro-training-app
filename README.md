# cellpro-training-app

新卒採用スキルテスト用のリポジトリです。

---

## お題：踏切シミュレーター

[`fumikiri-app/`](fumikiri-app/) に、ブラウザで動く **踏切シミュレーター** が入っています。

- 遮断機の開閉・警告灯の点滅・電車の走行アニメーション・効果音が連動するインタラクティブアプリです
- HTML / CSS / JavaScript（Phaser 3）のシングルファイル構成（`fumikiri-app/index.html`）で動作します
- 仕様の詳細は [`fumikiri-app/SPEC.md`](fumikiri-app/SPEC.md) を参照してください

---

## ルール

- **期限**：出題から **1週間以内** に提出
- **AI 利用推奨**（面白さ・スピード・品質が評価ポイント）
- 友人の協力は OK。ただし **自分で考えた部分・手伝ってもらった部分を明記すること**
- プログラミング言語は自由。ただし、環境構築の手順は採用担当が実行できるレベルで明記してください。
